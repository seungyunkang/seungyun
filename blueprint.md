
# 사주풀이 앱 Blueprint

## 개요

이 프로젝트는 사용자가 자신의 사주를 확인하고 다양한 운세를 볼 수 있는 웹 애플리케이션입니다. Firebase를 사용하여 호스팅 및 사용자 인증을 처리합니다.

## 구현된 기능

*   **Firebase Hosting:** 정적 웹사이트 호스팅을 위해 Firebase Hosting을 설정했습니다.
*   **기본 HTML 구조:** `index.html`에 기본적인 HTML 구조를 만들었습니다.
*   **기본 스타일:** `style.css`에 간단한 스타일을 추가했습니다.

## 현재 작업: Firebase 인증 구현

### 계획

1.  **HTML:**
    *   `index.html`에 회원가입, 로그인, 로그아웃을 위한 폼과 버튼을 추가합니다.
2.  **CSS:**
    *   `style.css`에 인증 관련 요소들의 스타일을 추가합니다.
3.  **JavaScript:**
    *   `main.js`에서 다음을 구현합니다.
        *   Firebase SDK를 초기화합니다.
        *   회원가입 폼 제출 시 Firebase에 신규 사용자를 생성하는 이벤트 리스너를 추가합니다.
        *   로그인 폼 제출 시 Firebase를 통해 사용자를 인증하는 이벤트 리스너를 추가합니다.
        *   로그아웃 버튼 클릭 시 사용자를 로그아웃시키는 이벤트 리스너를 추가합니다.
        *   사용자의 로그인 상태 변화를 감지하고 UI를 업데이트하는 `onAuthStateChanged` 리스너를 설정합니다.
